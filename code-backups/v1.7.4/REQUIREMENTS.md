# Requirements & Specifications — PaySick v1.7.4

**Version**: 1.7.4
**Date**: 2026-04-15

Carries forward all requirements from v1.7.3 with the following addition.

---

## New Requirements

### Direct Disbursement Mode

| ID | Requirement | Priority |
|----|-------------|----------|
| DISB-01 | All user-facing copy must reflect that PaySick disburses funds directly to the healthcare provider — no reference to a lender marketplace until external lenders are onboarded | Must Have |
| DISB-02 | The application review step must state that PaySick reviews the application and arranges disbursement | Must Have |
| DISB-03 | Post-submission messaging must confirm PaySick as the funding party, not a network of lenders | Must Have |

---
